## 📌 About the Project

This project contains a series of **CCNA-level network labs** designed to simulate small-to-medium business environments. Each lab focuses on a specific networking concept, from basic switch configuration to advanced dynamic routing protocols. All labs are built and tested using **Cisco Packet Tracer** and include detailed topology diagrams, configuration files, and verification steps.

<details>
<summary>📚 Click to expand: Lab Focus Areas</summary>

- ✅ VLANs & Inter-VLAN Routing (Router-on-a-stick)
- ✅ Static Routing & Default Routes
- ✅ Dynamic Routing (OSPFv2, EIGRP)
- ✅ STP, EtherChannel & Switch Security
- ✅ DHCP, NAT & ACLs
- ✅ IPv6 addressing & routing
- ✅ Network Troubleshooting scenarios

</details>
